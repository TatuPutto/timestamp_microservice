Timestamp microservice

Pass date (e.g. March 20, 2017) or Unix timestamp (e.g. 1489968000) as pathname, get Unix timestamp and natural language form of that date. If passed string is not valid date or Unix timestamp, return value for both properties is null.

Example usage:

https://blooming-fortress-68446.herokuapp.com/March-2-2017
https://blooming-fortress-68446.herokuapp.com/1489968000

Example output:

{"unix":1489788000,"natural":"March 18, 2017"}
